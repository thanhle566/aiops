Start the API.

## Usage

```
Usage: keep api [OPTIONS]
```

## Options
* `multi_tenant`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--multi-tenant`

  Enable multi-tenant mode


* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.



## CLI Help
