Create new config.

## Usage

```
Usage: keep config new [OPTIONS]...
```

## Options
* `interactive`:
  * Type: BOOL
  * Default: `True`
  * Usage: `--interactive`

  Create config interactively.

* `url`:
  * Type: STRING
  * Default: `http://localhost:8080`
  * Usage: `--url`

  The URL of the Keep backend server.

* `api-key`:
  * Type: STRING
  * Default: ``
  * Usage: `--api-key`

  The api key for authenticating over keep.

* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
