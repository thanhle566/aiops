sidebarTitle: "keep workflow apply"

Apply a workflow.

## Usage

```
Usage: keep workflow apply [OPTIONS]
```

## Options
* `file` (REQUIRED):
  * Type: Path
  * Default: `none`
  * Usage: `--file
-f`

  The workflow file


* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
