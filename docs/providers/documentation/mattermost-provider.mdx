## Inputs

The `notify` function takes the following parameters as inputs:

- `message`: Optional. The alert message to send to Mattermost.
- `blocks`: Optional. An array of blocks to format the message content.
- `channel`: Optional. The Mattermost channel to which the message should be sent.

## Outputs

N/A

## Authentication Parameters

The `MattermostProvider` requires the following authentication parameter:

- `webhook_url`: Required. Mattermost Webhook URL.

## Connecting with the Provider

1. **Obtain a Mattermost Webhook URL:**
   - Go to the Mattermost Incoming Webhook API documentation: [Mattermost Incoming Webhooks](https://docs.mattermost.com/developer/webhooks-incoming.html).
   - Follow the instructions to create a new incoming webhook.
   - Copy the generated webhook URL, which should be passed as the `webhook_url` for authentication.
